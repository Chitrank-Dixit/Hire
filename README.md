## 关于 BTC.com

BTC.com 团队服务于加密数字货币软件领域，产品线涵盖业界最顶尖的开源矿池、区块浏览器、钱包等数字货币相关服务。团队成员分布在北京、阿姆斯特丹、新加坡等地区。

BTC.com 隶属于北京比特大陆科技有限公司。北京比特大陆科技有限公司（ Bitmain ）是一家由集成电路技术团队、风险投资专家、企业家和资深互联网团队组成的高速成长中的高科技公司，总部位于北京，员工 1600 余人。研发中心分布在北京、深圳、成都、青岛、武汉、香港、台湾、美国旧金山、以色列特拉维夫、荷兰阿姆斯特丹、新加坡等地。公司同时具有软硬件和互联网相结合的研发能力，专注于全球领先水平高速运算芯片的研究，公司在比特币芯片行业已经做到全球第一，在数字货币运算领域占有全球70%以上市场份额。
